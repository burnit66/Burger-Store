# Burger-Store

Burger-Store uses persistant data to allow you to keep track of the burgers you've devoured. Add a new burger that you are eating, and then when you are done, click the devour button!

## Breakdown
Burger-Store uses:
 - Express
 - mySql
 - Handlebars
